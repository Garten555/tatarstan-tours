========================================
ИНСТРУКЦИЯ ПО СОЗДАНИЮ АНАЛИТИЧЕСКОЙ ЧАСТИ
========================================

1. УСТАНОВКА НЕОБХОДИМЫХ БИБЛИОТЕК
   
   Откройте командную строку и выполните:
   pip install python-docx

2. ЗАПУСК СКРИПТА
   
   Двойной клик по файлу: create_analytical_part.bat
   
   ИЛИ вручную:
   python create_analytical_part.py

3. РЕЗУЛЬТАТ
   
   Будет создан файл: АНАЛИТИЧЕСКАЯ_ЧАСТЬ.docx

4. ДОПОЛНИТЕЛЬНАЯ НАСТРОЙКА В WORD
   
   После создания документа откройте его в Microsoft Word и выполните:
   
   а) Настройка списков:
      - Выделите все списки
      - Правый клик -> "Изменить отступы в списке..."
      - Установите:
        * Положение номера: 1,25 см
        * Отступ текста: 0 см
        * Символ после номера: Пробел
   
   б) Проверка форматирования:
      - Заголовки глав: по центру, все прописные, интервал после 12 пт
      - Заголовки параграфов: с абзацным отступом 1,25 см
      - Основной текст: по ширине, абзацный отступ 1,25 см, междустрочный 1,5
      - Поля: левое 3 см, правое 1 см
   
   в) Вставка скриншотов:
      - В таблицах с пометкой "[ЗДЕСЬ ВСТАВИТЬ СКРИНШОТ]" вставьте скриншоты аналогов
      - Подписи к рисункам уже добавлены

5. ПРОВЕРКА
   
   Убедитесь, что:
   - Все разделы 1.1-1.5 присутствуют
   - В конце каждого раздела есть вывод
   - Таблицы для скриншотов заполнены
   - Форматирование соответствует ГОСТу
